## Використані технології

| Частина системи | Технологія |
| --- | --- |
| Web-застосунок | ASP.NET Core 10 |
| Робота з БД | Entity Framework Core |
| СУБД | PostgreSQL |

## Endpoints

| Endpoint | Призначення |
| --- | --- |
| `GET /` | повертає HTML-сторінку зі списком доступних бізнес-endpoints |
| `GET /health/alive` | перевіряє, що процес застосунку запущений |
| `GET /health/ready` | перевіряє готовність застосунку через підключення до PostgreSQL |


| Endpoint | Призначення |
| --- | --- |
| `GET /notes` | повертає список нотаток |
| `POST /notes` | створює нову нотатку |
| `GET /notes/{id}` | повертає повну інформацію про одну нотатку |

API підтримує два формати:

| Формат | Коли використовується |
| --- | --- |
| `application/json` | якщо клієнт очікує JSON |
| `text/html` | якщо клієнт передає `Accept: text/html` |
